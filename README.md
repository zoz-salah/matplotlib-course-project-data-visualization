# Data Visualization with Matplotlib

## Overview
This repository contains a complete beginner-friendly project for learning **data visualization using Matplotlib in Python**.  
The project is based on a 6-lesson course covering the most important and commonly used plot types.

Each lesson focuses on a specific visualization technique with clear examples and clean code.
